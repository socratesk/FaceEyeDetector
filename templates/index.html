Real-time Face and Eyes Detection Application


This application detects faces and eyes of one or more persons posed within the window, in real-time. This is developed and implemented using OpenCV, a real-time Open Source Computer Vision Library. Flask microframework is used for real-time video streaming that streams video from user's camera to server and back to user's browser after detecting object(s).
